## Description

A eslint config settings for [Zeraph](https://github.com/gentifly/zeraph) framework

## Installation

Yarn:

```sh
yarn add @gentifly/eslint-config@latest -D
```

NPM:

```sh
npm install @gentifly/eslint-config@latest -D
```
